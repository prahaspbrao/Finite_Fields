Introduction to Finite Fields
Definition
A finite field (Galois Field) is a field with a finite number of elements. It satisfies all field axioms: closure, associativity, commutativity, distributivity, identity elements, and inverses.
|F| = pn for some prime p
Field Properties
  • Closed under + and x
  • Associative and Commutative
  • Distributive law holds
  • Additive and Multiplicative identity
  • Every element has inverse
  • No zero divisors

Irreducible Polynomials
REDUCIBLE
Can be factored into polynomials of lower degree.
x2 + 1 over Z2
= (x + 1)2
IRREDUCIBLE
Cannot be factored further over the given field.
x2 + x + 1 over Z2
✓ Cannot be factored
WHY IMPORTANT?
Irreducible polynomials are the building blocks for constructing extension fields. They create quotient rings that become fields - enabling cryptographic systems, error-correcting codes, and advanced computing.

def list_field_elements(p, n, poly_coeffs):
    # Generate all coefficient combinations
    elements = []
    from itertools import product

    for coeffs in product(range(p), repeat=n):
        # Build polynomial representation
        terms = []
        for i, a in enumerate(coeffs):
            if a != 0:
                if i == 0:
                    terms.append(str(a))
                elif i == 1:
                    terms.append(f"{a}x" if a != 1 else "x")
                else:
                    terms.append(f"{a}x^{i}" if a != 1 else f"x^{i}")

        elem = " + ".join(terms) if terms else "0"
        elements.append(elem)

    return elements

Pollard's Rho Algorithm
KEY CONCEPT
A randomized factorization technique that uses a pseudo-random sequence and cycle detection (Floyd's algorithm) to find a non-trivial factor.
KEY FORMULA
d = gcd(|x - y|, n)
THE RHO CYCLE
Working of Pollard's Rho
1
Choose a random starting value x and constant c
2
Generate two sequences: x (slow) and y (fast, 2x speed)
3
Detect cycle using Floyd's algorithm: x moves 1 step, y moves 2 steps
4
Compute gcd(|x - y|, n) at each step
5
Extract factor when 1 < d < n (non-trivial factor found!)
xi+1 = (xi2 + c) mod n   |   yi+1 = f(f(yi))   |   d = gcd(|x - y|, n)

pollard_rho.py
import math, random

def pollard_rho(n):
    if n % 2 == 0: return 2
    x, y, c = random.randint(2, n-1), random.randint(2, n-1), random.randint(1, n-1)
    d = 1
    while d == 1:
        x = (pow(x, 2, n) + c) % n
        y = (pow(y, 2, n) + c) % n
        y = (pow(y, 2, n) + c) % n
        d = math.gcd(abs(x - y), n)
    return d if d != n else None

# Factor 8051
n = 8051
p = pollard_rho(n)
q = n // p
print(f"Factors: {p} x {q}")
Euler's Phi Function
DEFINITION
phi(n) counts the number of integers from 1 to n that are coprime to n.
FOR DISTINCT PRIMES
phi(n) = (p-1)(q-1)
phi(8051) = phi(83 x 97)
        = (83-1) x (97-1)
        = 82 x 96
        = 7872
IMPORTANCE IN RSA
Euler's totient is the cornerstone of RSA encryption:
  • Public key: (e, n)
  • Private key: (d, n)
  • e x d ≡ 1 (mod phi(n))
  • Encryption: c = m^e mod n
  • Decryption: m = c^d mod n
  • Security relies on factoring n
Without phi(n), RSA breaks!
